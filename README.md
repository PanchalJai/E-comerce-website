#  E-Commerce Website

##  Project Description
This project is developed as part of the **Corizo Internship requirement**.  
It is a basic **MERN Stack E-Commerce website** that includes authentication and CRUD functionalities.

The project demonstrates the core structure of an e-commerce platform.

---

##  Features
- Signup and Login functionality  
-  Role-based access (User / Admin)  
-  CRUD operations (Create, Read, Update, Delete)  
-  Basic shopping flow  
-  About Us page
